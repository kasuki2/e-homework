SOME és ANY HASZNÁLATA

Alapjában véve a some-ot kijelentő mondatokban, az any-t pedig kérdő és tagadó mondatokban használjuk.

A some nagyjábóli jelentése az, hogy néhány, valamennyi. Az any jelentését nehezebb meghatározni, gyakran nincs szó szerinti fordítása magyarban.

SOME
I have some money in my pocket.
Van némi pénz a zsebemben.
There are some people at the bus stop.
Van néhány ember a buszmegállóban.
I saw some good films last night.
Láttam néhány jó filmet tegnap este.
ANY
I don't have any ideas.
Nincs ötletem.
Do you have any information about Tom?
Van információd Tomról?
There isn't any information about Tom.
Nincs információ Tomról.

Mind a some és az any használható megszámlálható és megszámlálhatatlan főnevekkel. Csak annyi a különbség, hogy ha a főnév megszámlálható, akkor azt többes számba kell tenni.

Megszámlálható főnevek - többes számba kerülnek
There are some books on the shelf.
Van pár könyv a polcon.
Do you need any magazines?
Szükséged van magazinokra?
We don't have any dogs.
Nincs kutyánk.
Megszámlálhatatlan főnevek - csak egyes számban lehetnek
I want to drink some whisky.
Whiskyt akarok inni.
Is there any dust on the table again?
Megint poros az asztal?
I didn't use any soap.
Nem használtam szappant.

A megszámlálható főnevek esetében van még egy lehetőség. A some vagy az any helyett lehetséges a határozatlan névelő, a/an használata. Ilyenkor persze a főnév egyes számban áll.

Would you like a cat?
Szeretnél egy macskát?
There's a pen on my table.
Van egy toll az asztalomon.
I don't have a dog.
Nincs kutyám.
SOME használata kérdésekben

Ha megkínálunk valakit valamivel, vagy kérni szeretnénk valamit, akkor az gyakran egy kérdő mondattal valósul meg. Iyen esetekben, tehát kínáláskor és kéréskor, some-ot használhatunk egy tulajdonképpeni kérdő mondatban.

Would you like some cheese?
Szeretnél egy kis sajtot?
Can I have some beer, please?
Kaphatok egy kis sört?
Would you like some ice in your whisky?
Szeretnél egy kis jeget a whiskidbe?
Would you like some books?
Szeretnél pár könyvet?

ÖSSZEFOGLALÁS

Ha kijelentő mondattal van dolgunk, akkor some. Megszámlálható főneveknél többes szám!

There's some wine in this bottle.
There are some books on the shelf.

Kérdő és tagadó mondatokban any. Megszámlálható főneveknél többes szám!

Is there any wine in the bottle?
No, there isn't any wine in the bottle.
Are there any books on the shelf?
No, there aren't any books on the shelf.

Megszámlálható főnevek esetében a some vagy az any helyett a is állhat, ha azt akarjuk kifejezni, hogy csak egy dolog van vagy nincs:

I have a dog.
Do you have a dog?
I don't have a dog.
